# Tugas Keamanan Jaringan - Implementasi RSA (Streamlit)

Aplikasi berbasis web implementasi RSA

## 📝 Deskripsi
Program ini mendemonstrasikan proses enkripsi dan dekripsi pesan menggunakan algoritma Kriptografi Asimetris RSA (Rivest-Shamir-Adleman). Aplikasi ini akan melakukan *generate* Kunci Publik dan Kunci Privat, lalu menampilkan *output* dari setiap tahapan proses (Ciphertext & Deciphered Text) secara berurutan sesuai dengan instruksi tugas.

## 🛠️ Teknologi yang Digunakan
* Python 3
* Streamlit (Web Framework)
* Cryptography (Library RSA)
